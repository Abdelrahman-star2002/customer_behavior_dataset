🛒 Customer Shopping Behavior Analysis
📌 About This Project

In this project, I explored a dataset of around 3,900 customer purchases to better understand how people shop — what they buy, how much they spend, and how different factors like age, gender, and subscriptions affect their behavior.

The goal was to turn raw data into useful insights that could help a business make smarter decisions.

📊 What I Worked With

The dataset includes:

Customer information (age, gender, location, subscription status)
Purchase details (product, category, price, season, etc.)
Shopping behavior (discount usage, purchase frequency, ratings, shipping type)

There were a few missing values (especially in review ratings), which I handled during the cleaning phase.

🧹 Data Cleaning & Preparation

I started by cleaning and preparing the data using Python.

Loaded and explored the dataset using pandas
Fixed missing values by filling them with median values
Renamed columns to make them easier to work with
Created new features like age groups and purchase frequency
Removed unnecessary or duplicate information
Finally, I stored the cleaned data in a PostgreSQL database for further analysis
🔍 Data Analysis (SQL)

After preparing the data, I used SQL to answer real business questions like:

Who spends more: male or female customers?
Do people who use discounts still spend a lot?
Which products have the best ratings?
Do subscribers behave differently from non-subscribers?
Which customers are loyal vs new?
Which age group generates the most revenue?

This part helped me connect data analysis with real business thinking.

📈 Dashboard (Power BI)

To make the results easier to understand, I built an interactive dashboard in Power BI.

It shows:

Revenue trends
Customer segments
Product performance
Shopping patterns

This makes it easier to quickly explore insights without digging into raw data.

💡 Key Takeaways

Some interesting things I found:

Loyal and returning customers contribute a big part of revenue
Subscribers tend to spend more on average
Some products depend heavily on discounts
Different age groups have different shopping habits
